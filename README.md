# Star Wars Game – Python Turtle

![Python](https://img.shields.io/badge/Python-3.x-yellow?style=for-the-badge&logo=python)

## Visão Geral

Este projeto implementa um jogo no estilo *Star Wars* utilizando **Python Turtle**, combinando animação, orientação a objetos, cálculos matemáticos e controle de múltiplas entidades. A aplicação demonstra domínio sólido de técnicas frequentemente avaliadas por recrutadores em vagas que envolvem lógica, interatividade e fluência em Python.

## Importância do Projeto

O jogo reúne diversos pilares fundamentais do desenvolvimento de software:

### ● Estruturas de Controle e Lógica de Jogo  
Movimentação contínua, loops de atualização, eventos assíncronos e regras que coordenam player, inimigos, projéteis e pontuação.

### ● Programação Orientada a Objetos  
Uso de classes para estruturar entidades do jogo, garantindo organização, escalabilidade e clareza — um aspecto essencial em projetos profissionais.

### ● Matemática Aplicada  
Cálculo de ângulos com funções trigonométricas, detecção de colisões, vetores de direção e ajustes dinâmicos de velocidade, ilustrando domínio de lógica numérica.

### ● Manipulação Gráfica com Turtle  
Criação de formas customizadas, atualização de *frames*, movimentação controlada por eventos e uso de múltiplas instâncias de objetos renderizados em tempo real.

### ● Interação com Usuário  
O jogo implementa controles via teclado, escuta de eventos e resposta imediata ao jogador, elementos centrais em aplicações interativas.

## Funcionalidades Principais

- Sistema de nave espacial customizada com shape próprio  
- Inimigos com comportamento dinâmico e perseguição baseada em cálculo angular  
- Mísseis independentes com gerenciamento de estado  
- Sistema de pontos exibido em tempo real  
- Aumento progressivo de dificuldade  
- Detecção precisa de colisão  
- Estrutura modular, clara e reutilizável para outros jogos  

## Aplicações e Relevância Profissional

Este projeto demonstra competências valorizadas por recrutadores:

- Lógica avançada de programação  
- Capacidade de estruturar sistemas interativos  
- Compreensão de matemática aplicada  
- Organização orientada a objetos  
- Construção de ambientes gráficos e responsivos  
- Boas práticas no gerenciamento de múltiplas entidades
